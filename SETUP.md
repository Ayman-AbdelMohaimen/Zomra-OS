# 🚀 Zomra OS: Quick Start Guide

Welcome to the **Zomra OS** ecosystem. Follow these steps to get your local development environment running in minutes.

---

### 🛠️ Prerequisites
* **Node.js:** Version 18.x or higher.
* **Browser:** A modern Chromium-based browser (**Chrome, Edge, or Brave**). 
  > *Note: Safari and Firefox do not currently support the full File System Access API required for Zomra's local editing features.*

---

### ⚡ Installation Steps

1. **Clone the Kingdom:**
   ```bash
   git clone [https://github.com/Ayman-AbdelMohaimen/Zomra-OS.git](https://github.com/Ayman-AbdelMohaimen/Zomra-OS.git)
   cd Zomra-OS
````

2.  **Install Dependencies:**

    ```bash
    npm install
    ```

3.  **Launch the Engine:**

    ```bash
    npm run dev
    ```

4.  **Access the OS:**
    Open your browser and navigate to: [http://localhost:5173](https://www.google.com/search?q=http://localhost:5173)

-----

### 🔑 Activation & First Run

1.  **Initialize Workspace:**
    Click the **"Initialize Workspace"** button on the dashboard. Select any folder on your PC. This grants Zomra the authority to read and write code in that directory.

2.  **Connect the Brain (API Key):**

      - Go to **Settings** (Gear Icon).
      - Enter your `Gemini API Key`.
      - Select your preferred model (e.g., Gemini 1.5 Pro/Flash) to activate **Hermes**.

3.  **Verify the Swarm:**
    Check the **Insights Panel**; if you see the "Logic Trinity" pulsing, the swarm is ready to build.

-----

### 🛡️ Security & Privacy Note

Your **API Key** is stored strictly in your browser's `localStorage`. **Zomra OS** is a "Green-Code" privacy-first system; your keys and local code are never uploaded to any external server.
